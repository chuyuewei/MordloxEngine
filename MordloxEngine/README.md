# MordloxEngine

MordloxEngine 是一个基于C++20的现代游戏引擎，旨在提供高性能、易用性和灵活性。

## 功能特点

- 基于现代C++20
- 标准
- 跨平台支持
- 模块化架构
- 高性能渲染系统
- 物理引擎集成
- 资源管理系统
- 场景图管理
- 组件化实体系统

## 系统要求

- Visual Studio 2022 或更高版本
- C++14 兼容的编译器
